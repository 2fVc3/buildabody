🎪 Chaotic Jenga Circus 🎪

🎯 GOAL: Stack as many blocks as possible without toppling the tower!

📋 How to Play:

🎪 Start: Click "START THE CHAOS!" to begin
📱 Place Blocks: Click (or press spacebar) when the moving block is positioned where you want it
🏗️ Build Strategy: Each layer alternates direction - horizontal then vertical, like real Jenga!
⚖️ Balance: Blocks need at least 30% overlap with the layer below to stay stable
💥 Game Over: Tower falls if blocks don't have enough support
🎨 Chaos Effects: Some blocks have special effects - grow, shrink, speed up, slow down, or rainbow colors!

🏆 Scoring System

Your score = Number of blocks stacked

Base layer (3 blocks) doesn't count toward score

Each additional block you successfully place = +1 point

Compete with others on the leaderboard!
